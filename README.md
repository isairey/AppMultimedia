# 🌌 Acerca del proyecto

**React Media Streaming** es una plataforma multimedia enfocada en la exploración y reproducción de contenido audiovisual moderno.

La aplicación utiliza **React + Vite** para proporcionar una experiencia extremadamente rápida, fluida y optimizada.

El proyecto permite:

- 🎬 Explorar películas y series
- 🔥 Navegar contenido trending
- ⚡ Experiencia rápida y moderna
- 🌐 Streaming multimedia
- 📱 Diseño responsive
- 🎨 UI inspirada en plataformas OTT

# ⚡ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/your-user/react-media-streaming.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd react-media-streaming
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Ejecutar aplicación

```bash
npm run dev
```
